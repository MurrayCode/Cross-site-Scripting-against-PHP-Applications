Installation:

  1. Unzip the package in an empty directory and upload everything.
  2. Open sql.sql file and run it, it will create a database ULforum, and will create table under this database. You can also modify the name.
  3. Once the SQL query have been run, open the config.php and set the parameter to your needs, and also cahnge the connect.php too.
  4. Now you are ready to have the fun with ULforum.

System Requirements

  • PHP
  • MySQL
