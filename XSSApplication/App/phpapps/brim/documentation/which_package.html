Brim comes in two packages:

  • Brim-lite
  • Brim-full

Brim lite contains the following plugins:

  • Bookmarks
  • Calendar
  • Contacts
  • Notes
  • Tasks

Brim-full contains all the lite plugins and the following additional plugins:

  • Checkbook
  • Collections
  • E-banking
  • News/RSS
  • Passwords
  • Webtools

See the homepage for more details on the available plugins
